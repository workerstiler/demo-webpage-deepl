# Salon Site

Одностраничный сайт салона красоты.

## Структура проекта

```
salon-site/
├── index.html      — главная страница
├── css/
│   └── style.css   — стили
├── js/
│   └── main.js     — скрипты
├── images/         — фото для портфолио
└── README.md
```

## Запуск

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
npx serve .
```

## Портфолио

Поместите фотографии работ в папку `images/` и подключите их в секции «Портфолио» в `index.html`.
